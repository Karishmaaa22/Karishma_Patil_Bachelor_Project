This is College-department chatbot used for assisting department related queries along with navigation links of different department locations.
Additionally, it also saves user details such as phone number, name and queries asked. This solution was created for non-IT staff to analyze chatbot inputs and output.
Moreover, it does not require to refresh again and again it was working dyanmically.

On the other hand for admins, separate chabot service provider for allocated for chatbot using Google merchant services for providing emails, OTP's by chatbot to admin for accessing the backened services.

Frontend and Backend were totally protected and secured using web token services.

The chatbot was able to respond with text, images, web links, downloaded pdfs.

Technologies: MongoDB, Express, React, Node.js, Dialogflow, JavaScript
